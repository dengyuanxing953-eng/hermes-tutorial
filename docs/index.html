CHAPTER 04

VPN 和代理
你需要知道的

用国外AI服务，有时候需要"特殊的网络工具"。这一章帮你搞清楚。

直连被拦截 → 通过VPN/代理中转 → 成功访问AI服务器

🔐
什么是 VPN？
VPN = 虚拟专用网络。简单说就是给你的网络开一条加密隧道。

你的请求先走隧道到另一台服务器（通常在国外），再从那里访问目标网站。整个电脑的流量都走隧道。
🔀
什么是代理（Proxy）？
代理 = 一个"中间人"。你的请求先发给它，它帮你转发。

代理通常只管特定软件的流量，VPN管整个电脑。对AI开发来说，代理通常够用。
💡 关系：VPN是"专线"（整个电脑都走），代理是"转寄"（只管某个软件）。你装了VPN之后，代理也就有了——VPN客户端通常自带代理功能。

什么情况下需要？

✅ 需要翻墙
• 用 OpenAI（ChatGPT/Codex）的API
• 用 Anthropic（Claude）的API
• 用 Google Gemini 的API
• 访问 GitHub（偶尔慢）
• 下载国外软件
❌ 不需要翻墙
• DeepSeek — 国内直连 ✅
• 千问（阿里云）— 国内直连 ✅
• Kimi（月之暗面）— 国内直连 ✅
• MiMo（小米）— 国内直连 ✅
• 安装Hermes — 国内直连 ✅
⚠️ 重要：我们教程用国产API（DeepSeek/千问/Kimi/MiMo），不需要翻墙。但如果你想用Claude原版模型、ChatGPT等，就需要。建议先跟着教程走，以后需要再装。

为什么还是建议装一个？

🔓 Claude原版模型
Claude Code最强的编程能力来自Anthropic的Claude模型，需要翻墙才能接。后面会教你怎么接国产模型（不需要翻墙），也会教怎么接原版。
⬇️
GitHub下载加速
GitHub有时候国内很慢。有VPN下载快很多。

不过Hermes安装脚本有国内加速，装Hermes不需要翻墙。

怎么装？完整流程

1

买一个机场服务

机场 = 提供VPN/代理服务的商家。推荐我们用的这个：

👉 tolink.pro

注册时填邀请码：6aEXKBRj

月费大概 15-30元，按套餐选。付费的比免费的稳定得多。

2

下载代理客户端

买了机场之后，它会给你一个订阅链接和客户端下载地址。你需要装一个客户端来用它：

Windows：下载 Clash Verge（机场页面通常有下载链接，或GitHub搜"clash-verge"）
Mac：下载 ClashX Pro 或 Surge
iPhone：下载 Shadowrocket（小火箭，美区App Store，$2.99）
Android：下载 v2rayNG（Google Play或GitHub）

3

导入订阅，开启代理

① 打开客户端
② 找到"订阅"或"配置"选项
③ 粘贴机场给你的订阅链接，点导入/更新
④ 选一个节点（推荐选香港或日本，延迟低）
⑤ 点"开启"或"系统代理"

测试：浏览器打开 google.com，能打开就成功了。

4

让终端也走代理

开了代理后，浏览器能用了，但终端默认不走代理。需要设置：

MAC / LINUXexport https_proxy=http://127.0.0.1:7897
WINDOWS POWERSHELL$env:HTTPS_PROXY="http://127.0.0.1:7897"

7897 是 Clash Verge 默认端口，不同客户端可能不同（看客户端设置里的端口号）。

💡 永久生效：把上面的命令加到 ~/.bashrc（Mac/Linux）或设为Windows环境变量，这样每次开终端都有。具体方法见Claude Code配置那一章。

CHAPTER 05

开始安装

跟着 6 步走，30 分钟搞定。

6 步安装流程

1

安装 Hermes

打开你的终端：

Mac：按 Cmd+空格，输入"终端"，回车

Windows：先装Git（下面说），然后用Git Bash

⚠️ Windows用户必读：Hermes安装脚本是bash脚本，PowerShell不能直接跑。

你只需要做一件事：装Git。
① 按 Win+R，输入 winget install Git.Git 回车
② 或者打开微软商店，搜索 "Git"，安装
③ 装完后按 Win+R，输入 Git Bash，回车
④ 在Git Bash里运行下面的命令

在终端（Mac）或Git Bash（Windows）里粘贴：

curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
⏳ 安装需要几分钟，自动下载Python和依赖。看到 "Installation complete" 就成功。过程中问什么，一路回车。
2

启动并初始化

hermes setup

交互式向导，全部按回车用默认值。然后启动：

hermes

看到欢迎界面——它活了！输入 /exit 退出，去配API。

💡 "终端"是什么？文字版操作界面。不需要懂代码，复制粘贴就行。
3

注册API，拿钥匙

API = AI的"电费"。推荐 DeepSeek（最便宜）：

① 打开 platform.deepseek.com
② 手机号注册
③ 左侧「API Keys」→「创建API Key」→ 立刻复制保存到备忘录

⚠️ 钥匙只显示一次！复制后看不到。然后「充值」充 5~10块。

其他选择：

• 千问：dashscope.aliyun.com（有免费额度）
• Kimi：platform.moonshot.cn（有免费额度）
• MiMo：platform.mimo.xiaomi.com（推理能力强）

4

把钥匙配进Hermes

hermes model

① "Select provider" → 选DeepSeek
② "Enter API key" → 粘贴钥匙
③ "Select model" → 回车选默认

看到 "✓ Connected" 就成功了。

💡 验证：hermes chat -q "你好"，AI回复了就OK。
5

给助手一份"说明书"

方式一（推荐）：启动Hermes，直接跟它说：

请把你的行为规则改成这样：1）用中文回答 2）简洁明了 3）遇到报错先翻译成人话再给方案 4）我说"帮我装XXX"你就直接执行安装命令

方式二（更保险）：不启动Hermes，终端直接改：

hermes config set agent.system_prompt "你是一个中文AI助手。用中文回答，简洁明了。遇到报错先翻译成人话再给方案。用户说帮我装XXX就直接执行安装命令。"
6

让Hermes帮你装Claude Code

hermes

启动后说：

帮我安装 Claude Code，装完帮我验证一下

新开一个终端，输入 claude，看到欢迎界面就成功了！

⚠️ 需要Node.js。提示找不到node？去 nodejs.org 下LTS版装。或跟Hermes说"帮我装Node.js"。

CHAPTER 06

装一个好看的界面
跟 Hermes 聊天

Hermes 不只是命令行。它有一个网页版界面，跟 ChatGPT 一样——打字聊天、看历史、管理设置。装一个用起来更舒服。

1

安装 Web UI

在终端运行：

npm install -g hermes-web-ui
⚠️ 需要 Node.js。如果提示找不到npm，先装Node.js：nodejs.org 下LTS版。
2

启动 Web UI

先确保 Hermes Gateway 在运行：

hermes gateway start

然后启动 Web UI：

npx hermes-web-ui start

看到 "Server running on http://localhost:8648" 就成功了。

3

打开浏览器

打开浏览器，访问：

http://localhost:8648

默认账号：admin，密码：123456

登录后你就能看到一个好看的聊天界面——跟ChatGPT一样，左边是历史对话，右边是聊天窗口。在这里打字跟Hermes聊天，比命令行舒服多了。

💡 以后每次用Hermes：先开终端运行 hermes gateway start 和 npx hermes-web-ui start，然后打开浏览器访问 localhost:8648 就行。

CHAPTER 07

Claude Code
接入模型的具体流程

装好Claude Code后，你需要告诉它"用哪个大脑"。两种选择。

方案A：接Claude原版模型（需要翻墙+贵）

Claude原版编程能力最强，但API按量付费，比DeepSeek贵10-20倍，且需要翻墙。

A

注册 Anthropic，拿Key

① 打开 console.anthropic.com（需翻墙）
② 注册，进API Keys页面，创建Key，复制保存

B

配置环境变量

MAC / LINUXexport ANTHROPIC_API_KEY="sk-ant-你的钥匙"
WINDOWS POWERSHELL$env:ANTHROPIC_API_KEY="sk-ant-你的钥匙"

然后 claude 启动，自动用Claude模型。

方案B：接国产模型（不需要翻墙+便宜）

推荐这个方案。DeepSeek编程能力也很强，便宜10倍以上，国内直连。

B1

用 DeepSeek（推荐）

临时设置（当前窗口有效）：

MAC / LINUXexport ANTHROPIC_API_KEY="你的DeepSeek钥匙" export ANTHROPIC_BASE_URL="https://api.deepseek.com/v1"
WINDOWS POWERSHELL$env:ANTHROPIC_API_KEY="你的DeepSeek钥匙" $env:ANTHROPIC_BASE_URL="https://api.deepseek.com/v1"
💡 永久生效（重要！）：
Mac/Linux：
echo 'export ANTHROPIC_API_KEY="你的钥匙"' >> ~/.bashrc
echo 'export ANTHROPIC_BASE_URL="https://api.deepseek.com/v1"' >> ~/.bashrc
source ~/.bashrc

Windows PowerShell：
[Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY","你的钥匙","User")
[Environment]::SetEnvironmentVariable("ANTHROPIC_BASE_URL","https://api.deepseek.com/v1","User")
B2

用千问 / Kimi / MiMo

原理一样，URL不同：

千问export ANTHROPIC_BASE_URL="https://dashscope.aliyuncs.com/compatible-mode/v1"
KIMIexport ANTHROPIC_BASE_URL="https://api.moonshot.cn/v1"
MIMOexport ANTHROPIC_BASE_URL="https://api.xiaomimimo.com/v1"

API Key也换成对应平台的。

⚠️ 注意：写代码推荐 DeepSeek 或 MiMo。日常聊天 千问 和 Kimi 也不错。多试几个。

导出微信聊天记录

装好Hermes后，你可以用说话的方式让它帮你干活。比如——导出微信聊天记录，然后用AI分析。

💬
为什么要导出？
• 永久保存重要聊天记录
• 用AI分析内容——提取关键信息、待办、决策
• 生成聊天报告——频率、话题、谁说得多
• 导出后可以搜索、统计、做可视化
⚡
Hermes 一句话搞定
以前你需要手动下载好几个工具、输一堆命令。

现在有了Hermes，跟它说一句话，它帮你全部搞定。

怎么操作？

1

打开Hermes，跟它说

启动Hermes（终端输入 hermes，或打开Web UI），然后说：

帮我安装微信聊天记录导出工具。我需要导出微信聊天记录保存到本地，然后用AI分析。你帮我把需要的工具都装好，然后教我怎么用。我是 Windows/Mac 用户。
2

Hermes 会自动帮你做这些

① 下载 wx_key — 提取微信数据库密钥
② 下载 WDA（Windows Database Analyzer）— 解密微信数据库
③ 安装Python依赖 — 自动处理
④ 一步步指导你操作 — 告诉你点哪里、怎么导出

💡 Hermes会根据你的系统自动选择正确工具。Windows和Mac的微信数据库格式不同，它会自动处理。
3

导出完成后，让AI分析

继续跟Hermes说：

帮我分析刚才导出的聊天记录，生成一份报告：聊天频率、最常聊的话题、谁说得多、重要日期
⚠️ 隐私提醒：微信聊天记录属于个人隐私。导出后妥善保管，不要上传到公开地方。
💡 这就是智能体的威力。以前你需要搜教程、下工具、输命令。现在你说一句话，Hermes帮你搞定。它不只是给你答案，它帮你做事。
