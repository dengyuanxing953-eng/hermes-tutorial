CHAPTER 04

VPN 和代理
你需要知道的

用国外 AI 服务，有时候需要"特殊的网络工具"。搞清楚是什么、为什么需要、怎么弄。

直连被拦截 → 通过VPN/代理中转 → 成功访问AI服务器

🔐
什么是 VPN？
VPN = 虚拟专用网络。简单说就是给你的网络开一条加密隧道。

你的请求先走隧道到另一台服务器（通常在国外），再从那里访问目标网站。

常见工具：Clash Verge（Windows）、ClashX Pro（Mac）、小火箭（iPhone）
🔀
什么是代理（Proxy）？
代理 = 一个"中间人"。你的请求先发给它，它帮你转发。

代理通常只管特定软件的流量，VPN 管整个电脑的流量。

对 AI 开发来说，大多数时候代理就够用了。
💡 VPN vs 代理的关系：VPN 是"全身防护"（整个电脑的流量都走隧道），代理是"定点转发"（只管某个软件的流量）。
你可以理解为：VPN 是专线，代理是转寄。对用 AI 来说，代理通常就够了。

什么情况下需要？

✅ 需要翻墙
• 用 OpenAI（ChatGPT/Codex）的 API
• 用 Anthropic（Claude）的 API
• 用 Google Gemini 的 API
• 访问 GitHub（偶尔）
• 下载国外软件
❌ 不需要翻墙
• DeepSeek — 国内直连 ✅
• 千问（阿里云）— 国内直连 ✅
• Kimi（月之暗面）— 国内直连 ✅
• MiMo（小米）— 国内直连 ✅
• 安装 Hermes — 国内直连 ✅
⚠️ 重要：我们这个教程用的是国产 API（DeepSeek/千问/Kimi/MiMo），不需要翻墙，国内直接连。你现在可以放心往下走。

那为什么还要装 VPN / 机场？

虽然国产 API 够用，但有些场景你还是会需要：

🔓 用 Claude Code 的原版模型
Claude Code 默认用 Anthropic 的 Claude 模型。如果你想要最强的编程能力，需要接 Claude API——这个需要翻墙。

后面的教程会教你怎么接国产模型（不需要翻墙），也会教你怎么接 Claude 原版模型（需要翻墙）。
🌐 用 ChatGPT / Gemini 的网页版
如果你想直接用 ChatGPT、Gemini 的网页版聊天，需要翻墙。

但如果你只是用它们的 API（通过智能体调用），有些可以通过国内中转站访问。
⬇️
下载 GitHub 上的项目
GitHub 有时候在国内访问很慢或打不开。装了 VPN 后下载会快很多。

不过 Hermes 安装脚本已经做了国内加速，装 Hermes 不需要翻墙。
📡
未来可能需要
AI 领域发展很快，很多新工具、新模型都是国外出的。

有一个靠谱的 VPN / 机场在手，以备不时之需。

怎么弄？

这里不提供具体工具，但告诉你思路：

1

找一个靠谱的机场

搜索引擎搜"机场推荐 2026"，找评价好、稳定的服务。付费的比免费的稳定得多。月费大概 15-30 元。选的时候注意：要支持 Clash 或 V2Ray 协议的。

2

装客户端

Windows：下载 Clash Verge（GitHub 上搜）
Mac：下载 ClashX Pro
iPhone：下载 Shadowrocket（小火箭，美区 App Store，$2.99）
Android：下载 v2rayNG（Google Play 或 GitHub）

3

导入订阅链接

机场会给你一个订阅链接。在客户端里粘贴导入，选一个节点，开启代理。
测试：打开浏览器访问 google.com，能打开就成功了。

4

在终端里也用上代理

开了代理后，终端默认不走代理。需要设置环境变量：

MAC / LINUXexport https_proxy=http://127.0.0.1:7897
WINDOWS POWERSHELL$env:HTTPS_PROXY="http://127.0.0.1:7897"

7897 是 Clash Verge 默认端口，不同客户端可能不同。设置后终端里就能访问国外服务了。

CHAPTER 05

开始安装

跟着 6 步走，30 分钟搞定。

6 步安装流程

1

安装 Hermes

打开你的终端（Terminal）：

Mac：按 Cmd + 空格，输入"终端"，回车
Windows：按 Win + R，输入 powershell，回车

粘贴这行命令，按回车：

curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
💡 Windows 用户如果 curl 不可用：
① 打开微软商店，搜索 "Windows Terminal"，安装
② 用 Windows Terminal 重新打开，再试
③ 还是不行？先装 Git：winget install Git.Git，然后用 Git Bash 运行
⏳ 安装需要几分钟，它会自动下载 Python 和依赖。看到 "Installation complete" 就成功了。过程中问什么，一路回车。
2

启动并初始化

运行设置向导：

hermes setup

它会问你几个问题，全部按回车用默认值：

① "Select a provider" → 直接回车（后面再配）
② "Enable terminal tools?" → 回车（Yes）
③ 其他问题 → 一路回车

然后启动：

hermes

看到欢迎界面和输入框——它活了！

输入 /exit 退出，我们去配 API。

💡 "终端"是什么？终端是文字版的操作界面。平时用鼠标点来点去，在终端里用打字告诉电脑做什么。不需要懂代码，复制粘贴就行。
3

注册 API，拿钥匙

API = AI 的"电费"。按量付费，用多少花多少。推荐 DeepSeek（最便宜）：

① 打开 platform.deepseek.com
② 手机号注册登录
③ 左侧点 「API Keys」
④ 点 「创建 API Key」，起个名字（随便写）
⑤ 立刻复制，粘到备忘录

⚠️ 钥匙只显示一次！复制后就看不到。保存好。然后去「充值」充 5~10 块，够用很久。

其他选择：

• 千问：dashscope.aliyun.com（有免费额度）
• Kimi：platform.moonshot.cn（有免费额度）
• MiMo：platform.mimo.xiaomi.com（推理能力强）

4

把钥匙配进 Hermes

回到终端：

hermes model

交互式菜单：

① "Select provider" → 选 DeepSeek（或你注册的那个）
② "Enter API key" → 粘贴钥匙
③ "Select model" → 回车选默认

看到 "✓ Connected" 就成功了。

💡 验证一下：运行 hermes chat -q "你好"，AI 回复了就说明配好了。
5

给助手一份"说明书"

启动 Hermes：

hermes

直接跟它说话：

请把你的行为规则改成这样：1）用中文回答 2）简洁明了 3）遇到报错先翻译成人话再给方案 4）我说"帮我装XXX"你就直接执行安装命令

它会确认并保存。以后每次对话都遵守。

💡 这就是智能体：你用自然语言告诉它规则，它就照做。不需要改配置文件。
6

让 Hermes 帮你装 Claude Code

直接跟它说：

帮我安装 Claude Code，装完帮我验证一下

Hermes 会自动执行安装命令。弹出确认输入 y 回车。

新开一个终端窗口，输入：

claude

看到欢迎界面——你有两个 AI 助手了！

⚠️ 需要 Node.js。如果提示找不到 node，去 nodejs.org 下载 LTS 版安装。或者跟 Hermes 说"帮我装 Node.js"。

CHAPTER 06

Claude Code
接入模型的具体流程

装好 Claude Code 之后，你需要告诉它"用哪个大脑"。有两种选择：接 Claude 原版模型（需要翻墙），或接国产模型（不需要翻墙）。

方案 A：接 Claude 原版模型（需要翻墙）

Claude 原版模型是编程能力最强的选择。但需要 Anthropic 的 API Key，而且访问 API 需要翻墙。

A

注册 Anthropic，拿 API Key

① 打开 console.anthropic.com（需要翻墙）
② 注册账号，进入 API Keys 页面
③ 创建 Key，复制保存

⚠️ 需要翻墙才能访问。而且 Anthropic API 是按量付费，Claude 模型比较贵（比 DeepSeek 贵 10-20 倍）。
B

配置 Claude Code 使用 Anthropic API

在终端设置环境变量：

MAC / LINUXexport ANTHROPIC_API_KEY="你的钥匙"
WINDOWS POWERSHELL$env:ANTHROPIC_API_KEY="你的钥匙"

然后启动 Claude Code：

claude

它会自动使用 Anthropic 的 Claude 模型。输入 /model 可以查看当前模型。

💡 让钥匙永久生效：把 export ANTHROPIC_API_KEY="xxx" 加到 ~/.bashrc（Mac/Linux）或设置为 Windows 环境变量，这样每次打开终端都有。

方案 B：接国产模型（不需要翻墙）

如果你没有翻墙工具，或者想省钱，可以接国产模型。DeepSeek 的编程能力也很强，而且便宜 10 倍以上。

B1

用 DeepSeek（推荐，最便宜）

注册拿 Key（跟前面 Hermes 配 API 的步骤一样），然后：

MAC / LINUXexport ANTHROPIC_API_KEY="你的DeepSeek钥匙" export ANTHROPIC_BASE_URL="https://api.deepseek.com/v1"
WINDOWS POWERSHELL$env:ANTHROPIC_API_KEY="你的DeepSeek钥匙" $env:ANTHROPIC_BASE_URL="https://api.deepseek.com/v1"

启动 claude，它就会用 DeepSeek 模型了。

💡 原理：Claude Code 默认调用 Anthropic 的 API。我们通过设置 BASE_URL 把请求"转寄"到 DeepSeek 的服务器。DeepSeek 兼容 Anthropic 的接口格式，所以 Claude Code 能直接用。
B2

用千问 / Kimi / MiMo

原理一样，只是 URL 不同：

千问export ANTHROPIC_BASE_URL="https://dashscope.aliyuncs.com/compatible-mode/v1"
KIMIexport ANTHROPIC_BASE_URL="https://api.moonshot.cn/v1"
MIMOexport ANTHROPIC_BASE_URL="https://api.xiaomimimo.com/v1"

API Key 也换成对应平台的就行。

⚠️ 注意：不同模型能力不同。写代码推荐 DeepSeek 或 MiMo。日常聊天 千问 和 Kimi 也不错。可以多试几个。

• 把和重要的人的聊天记录永久保存
• 用 AI 分析聊天内容，提取关键信息、待办、决策
• 生成聊天报告——看看你们聊了什么、谁说得多
• 导出后可以搜索、统计、做可视化
⚡
Hermes 一键搞定
以前你需要手动下载好几个工具、输入一堆命令。

现在有了 Hermes，你只需要跟它说一句话，它帮你全部搞定。

怎么操作？

1

跟 Hermes 说

打开终端，启动 Hermes，然后说：

帮我安装微信聊天记录导出工具。我需要用它来导出微信的聊天记录，保存到本地，然后用 AI 分析。你帮我把需要的工具都装好，然后教我怎么用。

Hermes 会自动帮你做这些事

④ 告诉你怎么操作 — 一步步指导你导出

💡 Hermes 会根据你的系统自动选择正确的工具。Windows 和 Mac 的微信数据库格式不同，它会自动处理这些细节。
3

导出完成后

导出的聊天记录是 JSON 或 HTML 格式。你可以继续跟 Hermes 说：

Hermes 会读取文件、分析数据、生成报告。

⚠️ 注意：微信聊天记录属于个人隐私。导出后请妥善保管，不要上传到公开的地方。
💡 这就是智能体的威力。以前你需要自己搜教程、下工具、输命令。现在你只需要说一句话，Hermes 帮你搞定一切。它不只是给你答案，它帮你做事。
