FOR BEGINNERS

不会写代码，也行。跟着走就好。

Hermes 是什么？看 GitHub 介绍 →

STEP 1 / 6

安装 Hermes

打开你的终端：
Mac 在「应用程序 → 实用工具」找终端；
Windows 按 Win + R，输入 powershell 回车。

然后粘贴这行命令，回车：

curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

安装需要几分钟，看到 Installation complete 就成功了。
如果提示 curl: command not found，Windows 用户先装 curl for Windows。

STEP 2 / 6

启动它

在终端输入：

hermes setup

这是一个交互式向导，一路按回车用默认值就好。
它会帮你检查环境、创建配置文件。

设置完后，输入：

hermes

看到 Hermes 的欢迎界面，说明它活了！先输入 /exit 退出，我们去配 API。

STEP 4 / 6

拿钥匙，插进去

打开 platform.deepseek.com，手机号注册。

进「API Keys」→ 点「创建 API Key」→ 复制。

⚠️ 钥匙只显示一次。复制后立刻粘到备忘录。然后去「充值」充 5 块钱。

打开 dashscope.aliyun.com，用支付宝/淘宝登录。

进「API-KEY 管理」→ 点「创建」→ 复制。

💡 千问有免费额度，新用户送 100 万 tokens，不用充值就能开始。

打开 platform.moonshot.cn，手机号注册。

进「API Key 管理」→ 点「创建」→ 复制。

💡 新用户有免费额度，先体验再充值。

打开 platform.mimo.xiaomi.com，手机号注册。

进「API Keys」→ 点「创建」→ 复制。

💡 MiMo 推理能力强，代码任务表现好。新用户有免费额度。

👆 先在上一步选一个服务商，这里才会显示注册指引。

拿到钥匙后，在终端运行：

hermes model

按提示选你的服务商，粘贴 API Key，完成。

STEP 5 / 6

给助手一份说明书

这一步让 Hermes 变成你的专属助手。
在终端运行：

hermes config edit

找到 agent 那一栏，把下面这段粘进去作为 system_prompt：

你是一个友好的 AI 编程助手，名叫 Hermes。 ## 你的职责 - 帮助零基础用户学习编程和使用 AI 工具 - 回答用中文，简洁明了，少用专业术语 - 每次操作前先解释"这是什么"和"为什么要这样做" - 遇到报错，先翻译成人话，再给解决方案 ## 当用户说"帮我装 Claude Code" 1. 先确认用户的系统（Windows / Mac / Linux） 2. Windows：运行 winget install Anthropic.ClaudeCode 3. Mac：运行 brew install --cask claude-code@latest 4. Linux：运行 npm install -g @anthropic-ai/claude-code 5. 装完验证：运行 claude --version 6. 告诉用户：终端输入 claude 回车即可启动 ## 代码风格 - 写完代码解释每一行在干什么 - 优先用最简单的方案 - 出错不慌，一步步排查

不想手动编辑？也可以直接在 Hermes 对话里说：
"请把你的系统提示词改成：你是一个友好的 AI 编程助手……"

STEP 6 / 6

让它帮你装 Claude Code

重新启动 Hermes：

hermes

然后把这句话粘进去，回车：

帮我安装 Claude Code，装完帮我验证一下

Hermes 会自动帮你执行安装命令。
过程中如果弹出确认提示，输入 y 回车。

💡 装完之后，新开一个终端窗口，输入 claude 回车。
看到 Claude Code 的欢迎界面就成功了！
