大模型 vs 智能体
到底有什么区别？

你可能已经用过 ChatGPT、DeepSeek、Kimi。它们很聪明，但只能聊天——你问它答，仅此而已。

智能体（Agent）不一样。它能自己动手：帮你跑命令、装软件、改文件、操作浏览器、甚至写完代码直接部署上线。

CHAPTER 04

VPN 和代理
你可能需要知道这些

用国外的 AI 服务，有时候需要"特殊的网络工具"。这一章帮你搞清楚是什么、为什么需要、怎么弄。

🖼️
图片位置 ③：VPN/代理原理图
一条网络路径的可视化：
左边：用户的电脑（小房子图标）
中间：一个发光的"中转站"（标注 VPN / 代理）
右边：AI 服务器（云朵图标，标注 OpenAI / Anthropic）
普通路径：电脑→服务器（被一面墙挡住，标注"网络限制"）
VPN路径：电脑→中转站→服务器（绕过墙，畅通无阻）
暗色背景，路径用发光线条，科技风
🔐
什么是 VPN？
VPN 全称"虚拟专用网络"。简单说就是给你的网络开一条加密隧道。

你的网络请求先走隧道到达另一台服务器（通常在国外），再从那里访问目标网站。这样你就能访问一些在国内直接访问不了的服务。

常见工具：Clash、V2Ray、Shadowrocket（小火箭）
🔀
什么是代理（Proxy）？
代理是一个"中间人"。你的请求先发给它，它帮你转发到目标服务器。

代理和 VPN 很像，但代理通常只管特定软件的流量（比如只管浏览器），VPN 管整个电脑的流量。

在 AI 开发中，通常用代理就够了。
💡 它们什么关系？
VPN 是"全身防护"（整个电脑的流量都走），代理是"定点转发"（只管某个软件）。
对于用 AI 来说，大多数时候代理就够用了。你只需要让终端和 AI 工具能访问国外 API 就行。

哪些情况需要？

✅ 需要的场景
• 使用 OpenAI（ChatGPT/Codex）的 API
• 使用 Anthropic（Claude）的 API
• 使用 Google Gemini 的 API
• 访问 GitHub（偶尔需要）
• 安装某些国外软件
❌ 不需要的场景
• 使用 DeepSeek 的 API（国内直连）
• 使用 千问 的 API（阿里云直连）
• 使用 Kimi 的 API（国内直连）
• 使用 MiMo 的 API（小米直连）
• 安装 Hermes（国内直连）
⚠️ 重要提示：我们这个教程用的是国产 API（DeepSeek / 千问 / Kimi / MiMo），不需要翻墙，国内直接连。所以你现在可以放心往下走，不需要先搞定 VPN。

如果以后需要，怎么弄？

这里不提供具体工具，但告诉你思路：

1

找一个靠谱的机场/代理服务

搜索引擎搜"科学上网"、"机场推荐"，找评价好、稳定的服务。付费的通常比免费的稳定得多。月费大概 15-30 元。

2

装好客户端

大多数机场会给你一个订阅链接和客户端下载地址。Windows 用 Clash Verge，Mac 用 ClashX Pro 或 Surge，手机用小火箭（iOS）或 v2rayNG（Android）。

3

开启代理后，再用 AI 工具

代理开好之后，终端里设置一下环境变量（Hermes 的安装向导会帮你处理），就能正常访问国外 API 了。

CHAPTER 05

开始安装

准备好了？跟着下面 6 步走，30 分钟搞定。

🖼️
图片位置 ④：安装流程图
一条从左到右的时间线/进度条：
6个节点：安装Hermes → 启动 → 选API → 配置钥匙 → 设指令 → 装Claude Code
每个节点一个小图标（终端、火箭、钥匙、齿轮、对话框、完成勾）
暗色背景，发光线条连接，科技风，极简
1

安装 Hermes

首先，打开你的终端（Terminal）：

Mac 用户：按 Cmd + 空格，输入 终端，回车
Windows 用户：按 Win + R，输入 powershell，回车

然后粘贴下面这行命令，按回车：

curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
💡 Windows 用户注意：如果提示 curl 不是内部命令，说明你的 PowerShell 版本太老。解决方法：
① 打开微软商店（Microsoft Store），搜索 "Windows Terminal"，安装它
② 用 Windows Terminal 打开，再试一次上面的命令
③ 如果还是不行，先安装 Git：winget install Git.Git，然后用 Git Bash 运行上面的命令
⏳ 安装需要几分钟。它会自动下载 Python、安装依赖。看到 "Installation complete" 或类似的成功提示就OK了。
过程中如果问你什么，一路按 回车 用默认值就行。
2

启动并初始化

安装完成后，先运行设置向导：

hermes setup

它会问你几个问题，全部按回车用默认值就好：

① "Select a provider" → 直接回车（后面再配）
② "Enable terminal tools?" → 回车（选 Yes）
③ 其他问题 → 一路回车

设置完后，输入 hermes 启动：

hermes

你会看到一个欢迎界面和一个输入框——恭喜，它活了！

先输入 /exit 退出，我们去配 API。

💡 什么是终端？终端就是一个"文字版的操作界面"。你平时用鼠标点来点去，在终端里是用打字的方式告诉电脑做什么。不需要懂代码，跟着教程复制粘贴就行。
3

选一个 API，注册拿钥匙

API 就是 AI 的"电费"。你用 AI 回答问题、写代码，都需要调用 API。按量付费，用多少花多少。

推荐选 DeepSeek（最便宜）：

① 打开 platform.deepseek.com
② 用手机号注册登录
③ 左侧点 「API Keys」
④ 点 「创建 API Key」，给它起个名字（随便写）
⑤ 立刻复制，粘到备忘录里

⚠️ 钥匙只显示一次！复制之后就看不到了。一定要粘到备忘录保存好。
然后去「充值」页面充 5~10 块钱，按量计费，够用很久。

其他选择：

• 千问：dashscope.aliyun.com（有免费额度）
• Kimi：platform.moonshot.cn（新用户有免费额度）
• MiMo：platform.mimo.xiaomi.com（推理能力强）

4

把钥匙配进 Hermes

回到终端，运行模型配置：

hermes model

它会给你一个交互式菜单：

① "Select provider" → 选 DeepSeek（或你注册的那个）
② "Enter API key" → 粘贴你刚才复制的钥匙
③ "Select model" → 直接回车选默认的就行

配完后，它会自动测试连接。看到 "✓ Connected" 就成功了。

💡 怎么验证？运行 hermes chat -q "你好"，如果 AI 回复了你，说明配好了。
5

给助手一份"说明书"

这一步让 Hermes 变成你的专属助手，而不只是一个通用 AI。

在终端运行：

hermes

启动后，直接跟它说话：

请把你的行为规则改成这样：1）用中文回答 2）简洁明了 3）遇到报错先翻译成人话再给方案 4）我说"帮我装XXX"你就直接执行安装命令

它会确认并保存。以后每次对话都会遵守这些规则。

💡 这就是智能体的魅力：你用自然语言告诉它你想要什么，它就照做。不需要改配置文件，不需要懂代码。
6

让 Hermes 帮你装 Claude Code

现在，直接跟 Hermes 说：

帮我安装 Claude Code，装完帮我验证一下

Hermes 会自动帮你执行安装命令。过程中如果弹出确认，输入 y 回车。

装完之后，新开一个终端窗口，输入：

claude

看到 Claude Code 的欢迎界面——恭喜，你有两个 AI 助手了！

⚠️ Claude Code 需要 Node.js。如果提示找不到 node 命令，先装 Node.js：
去 nodejs.org 下载 LTS 版本，安装完重开终端再试。
（Hermes 也可以帮你装，跟它说"帮我装 Node.js"就行）
